Barky's Array Match

× Welcome to Array Match!

Barky just started programming classes and needs your help learning all about arrays.

Combine METHOD and ARGUMENT cards to change the board and match the provided solution.

Each row of the board represents an ARRAY.

Combine up one method card, and up to two argument cards to manipulate the board.

Reached a dead-end? Just press 'Restart Level' on the menu bar above!

Shift

Shift takes in ONE argument card.
Shift will target the row of the number you chose to combine.
Shift removes the first element of the row you have chosen and adds it to your hand.

Click here to learn more about Array.prototype.shift()! Unshift

Unshift takes in TWO argument cards.
Unshift will target the row of the first argument card you have chosen.
Unshift will move the number of the second argument card to the front of the row you have chosen.

Click here to learn more about Array.prototype.unshift()! Pop

Pop takes in ONE argument card.
Pop will target the row of the number you chose to combine.
Pop removes the last element of the row you have chosen and adds it to your hand.

Click here to learn more about Array.prototype.pop()! Push

Push takes in TWO argument cards.
Push will target the row of the first argument card you have chosen.
Push will move the number of the second argument card to the back of the row you have chosen.

Click here to learn more about Array.prototype.push()! Sort

Sort takes in ONE argument cards.
Sort will target the row of the number you chose to combine.
Sort will sort the row you have targeted in ascending order.

Click here to learn more about Array.prototype.sort()! Sum

Sum is a MATH card that takes in TWO argument cards.
Sum will add the TWO argument cards together and add the resulting value to your hand. Sub

Sub is a MATH card that takes in TWO argument cards.
Sub will subtract the TWO argument cards together and add the resulting value to your hand. Mod

Mod is a MATH card that takes in TWO argument cards.
Mod will return the remainder of the first number divided by the second number.
If the second number is greater than the first number, then the result will be the first number. Mult

Mult is a MATH card that takes in TWO argument cards.
Mult will multiply the TWO argument cards together and add the resulting value to your hand.
